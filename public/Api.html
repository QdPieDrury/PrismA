Welcome to the PrismA API documentation. Our service provides a simple way to create and manage temporary WebSocket rooms for your applications. All API responses are in JSON format.

Endpoints

GET /create-ws

Creates a new, unique WebSocket room. This is the first step to establishing a real-time connection. The room will automatically expire after 20 minutes of inactivity.

Response

{
  "success": true,
  "wsUrl": "wss://app.com/a42b1c7d-e8f9-4021-a123-b45678c90123",
  "id": "a42b1c7d-e8f9-4021-a123-b45678c90123",
  "disclaimer": "Keep this ID safe..."
}

Example with JavaScript

fetch('http://localhost:3000/create-ws')
  .then(response => response.json())
  .then(data => console.log(data));

Example with Python

import requests

response = requests.get('http://localhost:3000/create-ws')
print(response.json())

Example with Ruby

require 'net/http'
require 'json'

uri = URI('http://localhost:3000/create-ws')
response = Net::HTTP.get(uri)
puts JSON.parse(response)

Example with cURL

curl http://localhost:3000/create-ws
POST /close-ws/:id

Manually closes a specific WebSocket room and all its active connections. You must provide the room's unique ID in the URL.

Response

{
  "success": true,
  "message": "Room a42b1c7d-e8f9-4021-a123-b45678c90123 closed"
}

Example with JavaScript

const roomId = 'a42b1c7d-e8f9-4021-a123-b45678c90123';
fetch(`http://localhost:3000/close-ws/${roomId}`, {
  method: 'POST'
})
.then(response => response.json())
.then(data => console.log(data));

Example with Python

import requests

room_id = 'a42b1c7d-e8f9-4021-a123-b45678c90123'
url = f'http://localhost:3000/close-ws/{room_id}'
response = requests.post(url)
print(response.json())

Example with Ruby

require 'net/http'
require 'uri'
require 'json'

room_id = 'a42b1c7d-e8f9-4021-a123-b45678c90123'
uri = URI("http://localhost:3000/close-ws/#{room_id}")
http = Net::HTTP.new(uri.host, uri.port)
request = Net::HTTP::Post.new(uri.path)
response = http.request(request)
puts JSON.parse(response.body)

Example with cURL

curl -X POST http://localhost:3000/close-ws/a42b1c7d-e8f9-4021-a123-b45678c90123
